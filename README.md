# APT-dataset-limitation

This repository contains scripts for preprocessing techniques applied to recent APT datasets, including DAPT-2020, SCVIC-APT-2021, and Unraveled. The scripts also provide evaluation metrics such as Balanced Accuracy, F1-Score, TNR, and FPR results for models including DT, RF, SVM, GNB, kNN, and MLP.
